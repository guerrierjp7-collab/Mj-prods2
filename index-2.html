⚙️ Configuration Supabase
Suivez ces étapes pour activer la synchronisation de vos données sur tous vos appareils.
ÉTAPE 1 — CRÉER LE PROJET
Allez sur supabase.com → Créez un compte gratuit → Nouveau projet
Nommez votre projet mj-pro-cash, choisissez un mot de passe DB fort
Attendez ~2 min que le projet démarre
ÉTAPE 2 — CRÉER LES TABLES SQL
-- Copiez et collez dans SQL Editor de Supabase: CREATE TABLE IF NOT EXISTS reports ( id UUID DEFAULT gen_random_uuid() PRIMARY KEY, user_id UUID REFERENCES auth.users(id) ON DELETE CASCADE, branch TEXT NOT NULL DEFAULT 'ap', data JSONB NOT NULL DEFAULT '{}', label TEXT, updated_at TIMESTAMPTZ DEFAULT NOW(), UNIQUE(user_id, branch) ); ALTER TABLE reports ENABLE ROW LEVEL SECURITY; CREATE POLICY "Users manage own reports" ON reports FOR ALL USING (auth.uid() = user_id) WITH CHECK (auth.uid() = user_id);
ÉTAPE 3 — VOS CLÉS API
Clés disponibles dans : Settings → API → Project URL & anon key
